Place PDF files under this folder matching the file paths in data/catalog.json.
Example structure:
  /wwwroot/pdfs/class1/math.pdf
  /wwwroot/pdfs/class1/science.pdf

Notes about screenshot/screen-record prevention:
- Web apps cannot fully prevent OS-level screenshots or external screen recording.
- This app applies watermarking, hides/blur content when window loses focus or page is hidden, and disables context menu/selection to reduce casual copying.
- For stronger protection on Windows desktops, consider a native app using SetWindowDisplayAffinity.
